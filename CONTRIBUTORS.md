### How to Become a Contributor

1. **Report bugs** - Open issues for problems you find
2. **Submit fixes** - Create pull requests with improvements
3. **Improve docs** - Documentation improvements are always welcome
4. **Help others** - Answer questions in discussions and issues
5. **Share plugins** - Build and share plugins with the community

### Recognition

Contributors are recognized through:
- GitHub's contributor graph
- Release notes acknowledgments
- This file (for significant contributions)
